# Khadija Zubair Portfolio

Official portfolio website for Khadija Zubair, presenting work across UI/UX design, public speaking, and brand communication.

## Highlights

- Editorial, brand-forward landing experience
- Portfolio and speaking sections
- Reusable UI components and animated interactions
- Designed for modern responsive viewing across mobile and desktop

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- GSAP + Motion

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Run Locally

```bash
npm install
npm run dev
```

## Environment Setup

Use `.env.example` as your template:

```bash
cp .env.example .env.local
```

Fill in any required values before deployment.

## Build and Deploy

```bash
npm run build
npm run preview
```

Vercel helpers are included:

```bash
npm run vercel
npm run deploy
```
