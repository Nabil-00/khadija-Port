# Khadija Zubair Portfolio

Premium personal portfolio and speaking/design brand website for Khadija Zubair.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set required environment variables in `.env.local` (see `.env.example`)
3. Run the app:
   `npm run dev`
